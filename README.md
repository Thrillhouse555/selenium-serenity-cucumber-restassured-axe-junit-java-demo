# Selenium + Cucumber + Serenity BDD Test Suite

This is a demo of an automated test suiteusing Selenium WebDriver, Axe (by deque), RestAssured, Junit, Cucumber BDD, and Serenity reporting.

For this example the tests are testing my site https://georgeansell.co.uk

## Viewing Reports

Tests are run daily on Jenkins after which Serenity generates HTML reports which can be found here https://georgeansell.co.uk/serenity-reports/index.html
